# Hospital Bot

A TG bot for getting information about the hospital and its services.

## FEATURES

* FAQ
* Contacts
* About
* Driveway
* Work time

## DEPENDENCIES

* [aiogram](https://github.com/aiogram/aiogram)
* [python-dotenv](https://github.com/theskumar/python-dotenv)

## INSTALLATION

pip install -r requirements.txt
